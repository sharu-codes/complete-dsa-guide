# COMPLETE-DSA-GUIDE

A structured collection of Data Structures and Algorithms problems and solutions in multiple programming languages.

## Languages Covered

- [C](https://github.com/sharu-codes/complete-dsa-guide/tree/main/data-structures/c-programs) (ongoing)
- [Java](https://github.com/sharu-codes/complete-dsa-guide/tree/main/data-structures/java-programs) (ongoing)
- [Python](https://github.com/sharu-codes/complete-dsa-guide/tree/main/data-structures/python-programs) (ongoing)
- C++ (pending)

## Naming Convention

Each problem follows a numbered format:

```text
001_insertion_in_an_array.c
002_deletion_from_an_array.c
003_basic_sorting_of_an_array.c
...
```
